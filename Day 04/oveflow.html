CSS Overflow

When the content of an element overflows the defined boundaries we can handle them with the overflow property.

overflow-x and overflow-y properties are used tohandle the overflows only in a specific direction.

NOTE: The difference between overflow values scroll and auto is that scroll adds both (vertical and horizontal scroll bars) whereas auto adds only that is necessary.

overflow : visible;

This is a container of some text and it exceeds the predefined area. Here the overflow is visible.

overflow : hidden;

This is a container of some text and it exceeds the predefined area. Here the overflow is hidden.

overflow : scroll; (both vertical and horizontal scroll bars are included)

This is a container of some text and it exceeds the predefined area. Here the overflow is adjusted with a scroll bar.

overflow : auto; (only required scroll bar is included)

This is a container of some text and it exceeds the predefined area. Here also the overflow is adjusted with ascroll bar.
