CSS Inline-Block

The display : inline-block; makes an inline element to have the characteristics of a block level element but is still displayed as an inline element.

The main difference between an inline and inline-block is we cannot set top/bottom margins and padding, width and height but we can set them in inline-block.

The main difference between an block and inline-block is block elements are displayed each on a line whereas in inline-block elements are displayed on the same line.
